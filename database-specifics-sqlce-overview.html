Microsoft SQL Server Compact Edition
Programmer's Guide > Developer's Guide > Database Specifics > Microsoft SQL Server Compact Edition

Glossary Item Box

Microsoft SQL CE comes with the installation of Microsoft Visual Studio 2008 SP1. It can be used out of the box with Telerik OpenAccess ORM. All that should be specified unlike with the other database servers supported in a valid path to a database file. The database files used with Microsoft SQL CE have the .sdf extension. Microsoft SQL CE comes with a few restrictions in comparison with the other databases. The most important of them are the unability to use Stored Procedures, database Views or SQL queries that use a subquery as a scalar expression in the WHERE clause like this:
SELECT * FROM X WHERE (SELECT scalar_expression FROM Y) = B

Telerik OpenAccess ORM has been tested with the 3.5 version of Microsoft SQL Server Compact Edition.
